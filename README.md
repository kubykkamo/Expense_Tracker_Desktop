# 💰 Expense Tracker (WinForms Desktop)

> **⚠️ PROJECT STATUS: ARCHIVED / COMPLETED** > This desktop application has successfully reached its Minimum Viable Product (MVP) stage and is no longer in active development. The project is being entirely rebuilt as a modern Web Application to provide cross-platform accessibility and mobile support.

## 📌 About The Project
This is a lightweight, fully functional desktop application for tracking personal finances. It was originally built as a foundational project to master C#, Object-Oriented Programming (OOP), Windows Forms (WinForms) architecture, and local data persistence. 

While the app perfectly handles basic daily expense tracking, the desktop `.exe` format limits its usability on the go. Therefore, the core C# business logic is migrating to the web.

### ✨ Core Features (MVP)
* **Transaction Management:** Add, view, and organize daily income and expenses.
* **Custom Categories:** Create and assign custom categories to specific transactions.
* **Filtering & Data Binding:** Filter expenses dynamically by category directly in the data grid.
* **Local File Storage:** Persistent data saving mechanism without the need for an external SQL database.
* **Data Validation:** Built-in error handling and input verification to prevent corrupted entries.

## 🛠️ Built With
* **C#** * **.NET 10**
* **Windows Forms (WinForms)**
* LINQ (for data filtering and sorting)

## 🚀 What's Next?
The future of this project is a **Web Application** (ASP.NET Core / Blazor). The transition will allow users to log their expenses directly from their smartphones via a browser, eliminating the need for desktop installations and introducing a proper relational database (SQLite/EF Core).
